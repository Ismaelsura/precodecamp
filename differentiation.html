HTML TAG DIFFERENTIATION

Tag Similarity Differentiation
hr & br Empty tag and Separate of content hr shows horizontal line, br inserts a single line break
ul & ol Both are list ul are unordered list. ol are ordered list, ol has attributes
select & checkbox & radio button They are form input type Select has dropdown list, checkbox option to be selected, radio only one option
div & span Both define section in html document div is block level, span is inline level
script & style Both for embed other language script is used to embed javascript, style is used to embed css (inline)
